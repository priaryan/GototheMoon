# IMC Prosperity 3 Backtester Setup

To use the backtester, run the following command in this directory:

```bash
git clone https://github.com/jmerle/imc-prosperity-3-backtester.git
```

This will clone the external backtester into `imc-prosperity-3-backtester/` which is ignored by git to avoid duplicating the external repo.

Once cloned, you can run backtest simulations using the backtester's tools.
